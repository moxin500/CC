YOLO

Object detection algorithm

YOLO, an acronym for "You Only Look Once", is an object detection algorithm acclaimed for its speed and accuracy. We used YOLOv5 and YOLOv7 to train our program to recognize currency, menus, and price tags.

Python

Coding language

Python is the main coding language that we used to create our project. We utilized it throughout our project back-end to collect data and create and train our model.

JavaScript, CSS, and HTML

Coding languages

JavaScript, CSS, and HTML (HyperText Markup Language) are coding languages commonly used for website design. JavaScript assists in website interaction, CSS overlooks style, and HTML structures the layout of the page. We used these languages to code the website you are seeing right now!

Roboflow

Computer vision developer tool

Roboflow oversees the dataset by managing the images, annotations and labels, preprocessing, augmentations, file formats, one-click model training, deployment, and active learning. We easily stored, labeled, and augmented our dataset with Roboflow.
